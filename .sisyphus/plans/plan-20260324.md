## Context

User is taking an AI course and wants practical experience building agentic systems.

### Architecture: Manager → Worker Pattern
```
┌─────────────────────────────────────────────────────────────┐
│                      Manager Agent                           │
│                                                             │
│  • Receives task (process session notes)                    │
│  • Decides WHICH worker to spawn                           │
│  • Coordinates results                                     │
└─────────────────────────┬───────────────────────────────────┘
                          │ spawns
                          ▼
┌─────────────────────────────────────────────────────────────┐
│                  Note Creator Agent                          │
│              (Initial Worker - MVP Focus)                   │
│                                                             │
│  • Reads session notes                                     │
│  • Extracts entities (NPCs, Locales, Objects, Orgs)        │
│  • Decides WHAT TYPE of note to create                     │
│  • Generates Obsidian-compatible markdown                   │
│  • Writes to vault                                         │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                  Future Workers (v2+)                        │
│                                                             │
│  • Quest Worker - track quest progress                     │
│  • Combat Worker - log encounters                          │
│  • Loot Worker - item management                          │
│  • ...                                                     │
└─────────────────────────────────────────────────────────────┘
```
